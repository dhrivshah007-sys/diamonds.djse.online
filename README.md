# DJSÉ Diamonds Website

Welcome to the official website of DJSÉ Diamonds - specializing in premium CVD lab-created diamonds.

## Features

- **Product Showcase**: Browse our collections of diamond rings, necklaces, earrings, and custom designs
- **About Section**: Learn about our commitment to quality and sustainability
- **Blog**: Stay updated with the latest news and tips about diamonds
- **Contact Form**: Get in touch with our team for inquiries and custom orders

## Technology Stack

- HTML5
- CSS3
- JavaScript
- GitHub Pages (hosting)

## Setup

1. Clone this repository
2. No build process needed - it's a static website
3. Push to your GitHub repository to deploy on GitHub Pages

## Customization

- Edit `index.html` to update content
- Modify `styles.css` for styling changes
- Replace `logo.png` with your brand logo

## GitHub Pages Setup

1. Go to Settings → Pages
2. Select "Deploy from a branch"
3. Choose `main` branch
4. Your site will be live at `https://yourusername.github.io/djse-diamonds`

## Custom Domain (Cloudflare)

To use your custom domain:

1. Add DNS records in Cloudflare
2. Configure GitHub Pages to use your custom domain
3. Update the CNAME file if needed
